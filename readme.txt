Export you woocommerce ordered product it's billing address, shipping address and it's products info.


== Description ==
Export woocommerce order with you can filter perticular date period and it's status 
Two file format to export ( .xlsx, .csv, .json );

= How it Works =
* Select start date and end date to export
* Choose order status 
* Edit file name you want
* Choose file type
* Click export file will download

= Features =
* Select order date period
* Order status wise export
* Export multiple file type ( json, xlsx, csv )
* Export wc fields factory fields
* New order mail with order data attachment

= Documentation =


== Installation ==
1. Ensure you have latest version of WooCommerce plugin installed ( 2.2 or above )
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'woocommerce' menu in WordPress
